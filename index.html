Shoot That Note

By YYFun LLC

Intuitive; Learning while Playing !

Shoot That Note is designed to help music learners with better and faster sight-reading ability. Get the right notes to keep playing! You'll improve in no time once you start the game! No prior note-reading experience necessary. There are easy lessons to help you with. Start playing NOW and you'll soon recognize notes like a professional!
